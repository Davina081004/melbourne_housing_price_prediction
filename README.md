# Melbourne Housing Price Prediction

## Overview
A machine learning model that predicts Melbourne property sale prices 
across Hawthorn, Ringwood and Burwood using Linear Regression.

## Dataset
150 property sales collected from realestate.com.au across 3 suburbs.

## Tools Used
Python, pandas, numpy, scikit-learn, matplotlib, seaborn

## Steps
1. Data cleaning and preprocessing
2. Exploratory data analysis
3. Feature engineering
4. Linear Regression with 5-fold cross validation

## Results
- MAE: $202,197
- RMSE: $294,353
- R²: 0.676

## Key Findings
- Bedrooms and property type were the strongest predictors of price
- Ringwood had the most consistent pricing
- Hawthorn had the highest value outliers
